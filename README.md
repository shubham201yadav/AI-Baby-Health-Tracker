# AI Baby Health Tracker

AI Baby Health Tracker is a chatbot-based application designed to assist parents in monitoring infant health and providing basic care suggestions.

## 🚀 Features
- Interactive chatbot
- Health suggestions
- API integration

## 🛠️ Tech Stack
- Python / JavaScript
- API Integration
- Machine Learning

## 📌 How It Works
- User enters query
- Chatbot processes input
- API generates response
